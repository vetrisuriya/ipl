# IPL Utility Hub

A simple static website for IPL tools:

- IPL teams match table generator
- Net run rate calculator
- Strike rate calculator
- Bowling economy calculator

## Hosting on GitHub Pages

1. Create a GitHub repository and push this project.
2. In GitHub, go to `Settings` > `Pages`.
3. Select the branch to publish, such as `main` or `gh-pages`, and the root folder.
4. Save and wait for the GitHub Pages URL.

## Files

- `index.html` — landing page with links to all tools
- `teams.html` — teams match table generator
- `nrr.html` — net run rate calculator
- `strike.html` — strike rate calculator
- `economy.html` — bowling economy calculator
- `styles.css` — layout and design
- `script.js` — shared interactive scripts for all pages

## Usage

Open `index.html` in a browser or publish via GitHub Pages.
